## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/Xollonox/Chug.git
cd Chug

# Serve locally
python3 -m http.server 8000

# Open
open http://localhost:8000
```

> Or just **[play online](https://xollonox.github.io/Chug/)** — no setup needed.
