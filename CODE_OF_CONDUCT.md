# Code of Conduct

## Our Standard

This project should remain respectful, constructive, and welcoming.

Everyone participating in discussions, issues, pull requests, or other project spaces is expected to:

- Be respectful
- Give useful feedback
- Avoid harassment or personal attacks
- Stay focused on improving the project
- Respect the creative direction of Xollonox

## Unacceptable Behavior

Unacceptable behavior includes:

- Insults, harassment, threats, or bullying
- Spam or intentionally disruptive comments
- Discrimination of any kind
- Sharing private information without permission
- Demanding changes in a hostile or disrespectful way

## Enforcement

Xollonox may remove comments, close issues, reject pull requests, or block users who violate this code of conduct.

## Scope

This code applies to all official project spaces, including GitHub issues, pull requests, discussions, and related community areas.
