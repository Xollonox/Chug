# Security Policy

## Supported Project

This repository contains a static browser game made with HTML, CSS, and JavaScript.

The currently supported version is the latest code on the `main` branch.

## Reporting a Security Issue

If you discover a security problem, please report it responsibly by opening a private contact path with the repository owner where possible, or by creating a GitHub issue without exposing sensitive exploit details.

Please include:

- A clear description of the issue
- Steps to reproduce it
- Browser/device information if relevant
- The possible impact

## Scope

Relevant security issues may include:

- Unsafe script injection risks
- Dangerous external resource behavior
- Malicious dependency suggestions
- GitHub Pages deployment problems
- Any issue that could compromise users opening the game page

## Out of Scope

The following are usually not security issues for this project:

- Gameplay balance problems
- Visual bugs
- Browser compatibility quirks without security impact
- Local modifications made outside the official repository
